# このガイドの使い方

* 左のサイドバーから目次にアクセスできます。
* 多くのページには操作できるシーンがあります。 もしシーンに ![プラス](assets/diagrams/plus.png)
と ![マイナス](assets/diagrams/minus.png) のズームボタンがあったら、カメラの回転、移動ができます。
左クリックを押しながらドラッグで回転、右クリックを押しながらドラッグすると平行移動します。

# Applied Energistics 2とは？

Applied Energistics 2は、ストレージと物流のための機械やコンポーネントを追加します。
部屋いっぱいのチェストはコンパクトなMEネットワークとなります。しかしこれは始まりに過ぎないのです。
Applied Energisticsは、modpackや他のmodと連携して自動化することを目的としています。 
1クリックですべての複雑な中間素材（またはアイテム）をクラフト、拠点の在庫管理をすることができます！
